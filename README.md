# VYN Player

A fast, minimal, and beautifully designed offline music player focused on simplicity and smooth performance.

VYN Player is built for users who prefer a clean listening experience without ads, distractions, or unnecessary complexity.

---

## ✨ Features

- Offline music playback
- Lightweight and optimized performance
- Clean and modern UI
- No Ads, No Tracking
- Smooth navigation and playback experience
- Built-in OTA update system

---

## ⚡ Performance

Designed to run smoothly across all devices, from low-end to high refresh rate screens, ensuring consistent and lag-free interaction.

---

## 🎯 Philosophy

VYN Player focuses on what actually matters:
- Speed
- Simplicity
- Stability

No unnecessary features. Just pure music.
